Информация о командах
.help : Открывает страницу с подсказками.

.offline : Поставить персонажа на offline торговлю/крафт.

.repair : Repairing character located in current account

.whoiam : Показать все характеристики персонажа.

.hellbound : Показать всю информацию про остров ада.

.cfg : Персональные настройки персонажа.

.premium : Информация о премиум аккаунте.

.divorce : Украшения для свадьбы.

.gotolove : Телепорт к своему мужу или жене.

.km-all-to-me : Призвать всех сокланов.



/clanpenalty : Показать информацию про штрафы клана.

/attacklist : Показать кланы, которым обьявлена война.

/underattacklist : Показать кланы, которые обьявили войну

/warlist : Показать кланы, которые в состоянии войны.

/instancezone : Показать время до повторного входа в инстанс.

/olympiadstat : Показать статистику олимпиады персонажа.

/mybirthday : Информация о дне рождения персонажа.

/unstuck : Телепортирует персонажа в ближайший город.

/channelcreate : Создать канал команды.

/channeldelete : Удалить канал команды.

/channelinvite : Пригласить игрока в канал команды.

/channelkick : Исключить игрока с канала команды.

/channelleave : Выйти с канала команды.
