Помощник Тестового Сервера:
Для того, чтобы Камаэль стал Дворянином, необходимо выполнить Квест Семена Хаоса. Возьмите Звезду Судьбы и встретьтесь с Иерархом Кекропусом.
Для выполнения Семен Хаоса, необходима Звезда Судьбы. Приобретите его с помощью функции "Купить Квестовый Предмет".
После выполнения квеста "Семена Хаоса", вы сможете стать Дворянином, воспользовавшись функцией в диалоге Помощника Тестового Сервера.
