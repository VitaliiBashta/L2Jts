Темный Маг - смена профессии:
Чтобы стать Заклинателем Ветра, игрок должен успешно пройти три квеста. Испытание мудрости может выполнять только персонаж, достигший 35-го уровня. Этот квест начинается со встречи с Магистром Мириен из Диона. Испытание судьбы доступно только по достижении 37-го уровня. Поговорите с Магистром Кайрой из Гирана. Наконец, Испытание мага можно проходить только начиная с 39-го уровня. Найдите Барда Рукала из Диона.
