Темный Рыцарь - смена профессии:
Чтобы стать Танцором Смерти, нужно успешно пройти три квеста. Испытание храбрости доступно, начиная с 35-го уровня. Этот квест начинается со встречи с Кэшем в Дионе. Испытание судьбы можно проходить с 37-го уровня. Для начала надо поговорить с Магистром Кайрой из Гирана. Наконец, для Испытания силы необходим 39-й уровень. Найдите в Орене Дуэлиста Каена.
