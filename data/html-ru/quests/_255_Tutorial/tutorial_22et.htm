Светлый Рыцарь - смена профессии:
Чтобы стать Рыцарем Евы, необходимо успешно пройти три квеста. Испытание верности доступно только по достижении 35-го уровня. Этот квест начинается со встречи с Великим Мастером Ханнавальтом из Гирана. Испытание жизни может проходить только игрок, достигший 37-го уровня. Для начала квеста поговорите с Мастером Кардиен из Диона. Наконец, Испытание целителя доступно только персонажам, достигшим 39-го уровня. Найдите в Гиране Жреца Банделлоса.
