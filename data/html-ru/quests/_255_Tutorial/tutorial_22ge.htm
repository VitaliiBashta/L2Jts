Светлый Маг - смена профессии:
Чтобы стать Призывателем, Светлый Маг должен успешно пройти три квеста. Испытание мудрости может выполнять только персонаж, достигший 35-го уровня. Этот квест начинается со встречи с Магистром Мириен из Диона. Испытание жизни доступно только по достижении 37-го уровня. Поговорите с Мастером Кардиен из Диона. Наконец, Испытание Призывателя может проходить только игрок, достигший 39-го уровня. Найдите Великого Призывателя Галатею из Глудина.
