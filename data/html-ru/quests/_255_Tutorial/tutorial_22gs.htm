Светлый Маг - смена профессии:
Чтобы стать Певцом Заклинаний, персонаж должен успешно пройти три квеста. Испытание мудрости доступно только по достижении 35-го уровня. Этот квест начинается со встречи с Магистром Мириен из Диона. Испытание жизни может проходить только персонаж, достигший 37-го уровня. Поговорите с Мастером Кардиен из Диона. Наконец, Испытание магии доступно только на 39-м уровне или выше. Найдите Барда Рукала из Диона.
