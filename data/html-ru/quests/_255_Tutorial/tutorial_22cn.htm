Маг - смена профессии:
Чтобы стать Некромантом, Вы должны успешно пройти три квеста. Испытание мудрости доступно персонажу 35-го уровня и выше. Этот квест начинается со встречи с Магистром Мириен в Дионе. Испытание доверия может проходить только игрок, достигший 37-го уровня. Для начала квеста поговорите с Верховным Жрецом Холлинтом из Орена. Наконец, Испытание колдовства можно пройти только начиная с 39-го уровня. Найдите Орима Тень у Академии Хардина.
