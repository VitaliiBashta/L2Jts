Темный Маг - смена профессии:
Чтобы стать Последователем Тьмы, игрок должен успешно пройти три квеста. Испытание мудрости можно проходить только по достижении 35-го уровня. Этот квест начинается со встречи с Магистром Мириен из Диона. Испытание судьбы доступно только начиная с 37-го уровня. Поговорите с Магистром Кайрой из Гирана. Наконец, Испытание призывателя может проходить только игрок, достигший 39-го уровня. Найдите Великого Призывателя Галатею из Глудина.
