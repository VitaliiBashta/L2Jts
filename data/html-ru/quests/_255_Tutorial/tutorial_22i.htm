Налетчик - смена профессии:
Налетчик, достигший 40-го уровня, может снова сменить профессию и стать Разрушителем. Для этого необходимо успешно пройти три квеста. Испытание храбрости доступно, начиная с 35-го уровня. Этот квест начинается со встречи с Кэшем в Дионе. Испытание славы можно проходить с 37-го уровня. Для начала надо поговорить с Префектом Вокианом из Гирана. Наконец, для Испытания мастерства необходим 39-й уровень. Найдите в Гиране Ветерана Аскалона.
